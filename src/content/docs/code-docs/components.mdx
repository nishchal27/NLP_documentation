# Components Module

The Components module contains reusable components for the SAAS web app "NLP". These components are categorized under different sections such as forms, global components, icons, media, sidebar, site navigation, UI, and unauthorized access handling.


## <span class="comp-section-title" id="overview">Overview</span>

The Components module is organized into various subdirectories, each containing related components that serve specific purposes. These components enhance the functionality and user experience of the application.
